3. 无重复字符的最长子串

难度: Medium | 标签: Hash Table String Sliding Window

归类: 1. 滑动窗口与双指针 > Hash Table

 题目描述

给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

 

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

 

提示：

	0 <= s.length <= 5  104
	s 由英文字母、数字、符号和空格组成

---
 解题思路与复盘

 一句话直击本质
使用滑动窗口和哈希集合来动态维护一个无重复字符的子串，并在遍历过程中更新最长子串的长度。

 综合思路
1. 滑动窗口：通过两个指针（left和right）来定义一个窗口，窗口内的字符通过一个列表或集合来维护。当窗口内出现重复字符时，移动左指针以缩小窗口，直到窗口内无重复字符。
2. 哈希集合：使用集合来快速判断字符是否重复，集合的查找和删除操作都是平均 O(1) 的时间复杂度。
3. 双指针：两个指针分别表示当前子串的起始和结束位置，通过移动右指针扩展子串，移动左指针缩小子串，确保子串内无重复字符。

 全量伪代码
plaintext
函数 lengthOfLongestSubstring(字符串 s):
    将字符串 s 转换为字符列表 string
    初始化 size 为 string 的长度
    初始化 left 和 right 指针为 0
    初始化 res 列表为空
    初始化 maxLen 为 0

    当 right 小于 size 时:
        将 string[right] 添加到 res 列表
        将 right 增加 1

        当 res 列表的长度不等于 res 集合的长度时:
            删除 res 列表的第一个元素
            将 left 增加 1

        更新 maxLen 为 max(maxLen, right-left)

    返回 maxLen

 复杂度
- 时间复杂度：O(n)，其中 n 是字符串的长度。每个字符在最坏情况下会被访问两次（一次通过右指针，一次通过左指针）。
- 空间复杂度：O(min(n, m))，其中 m 是字符集的大小，主要用于存储当前窗口内的字符。
