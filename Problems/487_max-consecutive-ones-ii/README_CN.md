# 487. 最大连续1的个数 II

**难度**: Medium | **标签**: `Array` `Dynamic Programming` `Sliding Window`

## 题目描述

None

## 解题思路与复盘

1. 一句话直击本质：使用滑动窗口技术，在遇到超过一个零时调整窗口的左边界，以保持窗口内最多只有一个零，从而计算最大连续1的长度。

2. 综合思路：
   - 滑动窗口：通过维护一个动态窗口，使用两个指针（左指针和右指针）来遍历数组，并记录窗口内0的数量，当窗口内0的数量超过1时，移动左指针以缩小窗口，确保窗口内最多只有一个0。
   - 计数器：使用字典或数组来记录窗口内0和1的数量，以便快速判断窗口内0的数量是否超过1。

3. 全量伪代码：
   ```
   初始化计数器 cnt 为一个空字典
   初始化左指针 left 为 0
   初始化右指针 right 为 0
   初始化最大长度 maxLen 为 0

   当右指针小于数组长度时：
       将 nums[right] 的计数加1
       将右指针右移一位

       当计数器中0的数量大于1时：
           将 nums[left] 的计数减1
           将左指针右移一位

       更新 maxLen 为 max(maxLen, right - left)

   返回 maxLen
   ```

4. 复杂度：
   - 时间复杂度：$O(n)$，其中 $n$ 是数组的长度，因为每个元素最多被访问两次（一次由右指针，一次由左指针）。
   - 空间复杂度：$O(1)$，因为计数器的大小是固定的（最多包含两个键：0和1）。
