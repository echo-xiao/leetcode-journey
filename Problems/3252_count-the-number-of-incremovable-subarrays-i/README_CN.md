# 3252. 统计移除递增子数组的数目 I

**难度**: Easy | **标签**: `Array` `Two Pointers` `Binary Search` `Enumeration`

## 题目描述

<p>给你一个下标从 <strong>0</strong>&nbsp;开始的 <b>正</b>&nbsp;整数数组&nbsp;<code>nums</code>&nbsp;。</p>

<p>如果 <code>nums</code>&nbsp;的一个子数组满足：移除这个子数组后剩余元素 <strong>严格递增</strong>&nbsp;，那么我们称这个子数组为 <strong>移除递增</strong>&nbsp;子数组。比方说，<code>[5, 3, 4, 6, 7]</code>&nbsp;中的 <code>[3, 4]</code>&nbsp;是一个移除递增子数组，因为移除该子数组后，<code>[5, 3, 4, 6, 7]</code>&nbsp;变为&nbsp;<code>[5, 6, 7]</code>&nbsp;，是严格递增的。</p>

<p>请你返回 <code>nums</code>&nbsp;中 <b>移除递增</b>&nbsp;子数组的总数目。</p>

<p><b>注意</b>&nbsp;，剩余元素为空的数组也视为是递增的。</p>

<p><strong>子数组</strong> 指的是一个数组中一段非空且连续的元素序列。</p>

<p>&nbsp;</p>

<p><strong class="example">示例 1：</strong></p>

<pre>
<b>输入：</b>nums = [1,2,3,4]
<b>输出：</b>10
<b>解释：</b>10 个移除递增子数组分别为：[1], [2], [3], [4], [1,2], [2,3], [3,4], [1,2,3], [2,3,4] 和 [1,2,3,4]。移除任意一个子数组后，剩余元素都是递增的。注意，空数组不是移除递增子数组。
</pre>

<p><strong class="example">示例 2：</strong></p>

<pre>
<b>输入：</b>nums = [6,5,7,8]
<b>输出：</b>7
<b>解释：</b>7<strong>&nbsp;</strong>个移除递增子数组分别为：[5], [6], [5,7], [6,5], [5,7,8], [6,5,7] 和 [6,5,7,8] 。
nums 中只有这 7 个移除递增子数组。
</pre>

<p><strong class="example">示例 3：</strong></p>

<pre>
<b>输入：</b>nums = [8,7,6,6]
<b>输出：</b>3
<b>解释：</b>3 个移除递增子数组分别为：[8,7,6], [7,6,6] 和 [8,7,6,6] 。注意 [8,7] 不是移除递增子数组因为移除 [8,7] 后 nums 变为 [6,6] ，它不是严格递增的。
</pre>

<p>&nbsp;</p>

<p><strong>提示：</strong></p>

<ul>
	<li><code>1 &lt;= nums.length &lt;= 50</code></li>
	<li><code>1 &lt;= nums[i] &lt;= 50</code></li>
</ul>

## 解题思路与复盘

1. 一句话直击本质：算法的核心逻辑是通过识别数组中不满足递增条件的边界，计算可以移除的递增子数组的数量。

2. 综合思路：
   - **双指针法**：通过两个指针分别从数组的两端向中间移动，找到第一个不满足递增条件的元素，然后通过二分查找或线性扫描计算可移除的递增子数组数量。
   - **暴力法**：枚举所有可能的子数组，检查每个子数组是否为递增的。

3. 全量伪代码：
   - **双指针法**：
     ```
     初始化 first 为数组长度减一
     从左到右遍历数组，找到第一个不满足递增条件的位置，更新 first
     初始化 last 为 0
     从右到左遍历数组，找到第一个不满足递增条件的位置，更新 last
     如果 first 等于数组长度减一，返回数组长度乘以数组长度加一除以二
     初始化 ans 为数组长度减去 last 加一
     从 0 到 first 遍历：
         初始化 left 为 last，right 为数组长度减一
         二分查找：
             如果中间元素大于当前元素，更新 right
             否则，更新 left
         更新 ans 为 ans 加上数组长度减去 left 加一
     返回 ans
     ```
   - **暴力法**：
     ```
     初始化 cnt 为 0
     从 0 到数组长度遍历 i：
         从 i 到数组长度遍历 j：
             移除第 i 到 j 的子数组
             如果剩余数组递增，cnt 加一
     返回 cnt
     ```
   - **双指针法（另一种实现）**：
     ```
     初始化 l 为 0，r 为数组长度减一
     从左到右找到第一个不满足递增条件的位置，更新 l
     从右到左找到第一个不满足递增条件的位置，更新 r
     如果 l 等于数组长度减一，返回数组长度乘以数组长度加一除以二
     初始化 j 为 r，cnt 为数组长度减去 r 加一
     从 0 到 l 遍历 i：
         当 j 小于数组长度且当前元素大于等于 j 位置的元素时，j 加一
         更新 cnt 为 cnt 加上数组长度减去 j 加一
     返回 cnt
     ```

4. 复杂度：
   - **双指针法**：时间复杂度为 $O(n \log n)$，空间复杂度为 $O(1)$。
   - **暴力法**：时间复杂度为 $O(n^3)$，空间复杂度为 $O(n)$。
