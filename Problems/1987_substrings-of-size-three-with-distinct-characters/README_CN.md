# 1987. 长度为三且各字符不同的子字符串

**难度**: Easy | **标签**: `Hash Table` `String` `Sliding Window` `Counting`

## 题目描述

<p>如果一个字符串不含有任何重复字符，我们称这个字符串为 <strong>好</strong> 字符串。</p>

<p>给你一个字符串 <code>s</code> ，请你返回 <code>s</code> 中长度为 <strong>3</strong> 的 <strong>好子字符串</strong> 的数量。</p>

<p>注意，如果相同的好子字符串出现多次，每一次都应该被记入答案之中。</p>

<p><strong>子字符串</strong> 是一个字符串中连续的字符序列。</p>

<p> </p>

<p><strong>示例 1：</strong></p>

<pre>
<b>输入：</b>s = "xyzzaz"
<b>输出：</b>1
<b>解释：</b>总共有 4 个长度为 3 的子字符串："xyz"，"yzz"，"zza" 和 "zaz" 。
唯一的长度为 3 的好子字符串是 "xyz" 。
</pre>

<p><strong>示例 2：</strong></p>

<pre>
<b>输入：</b>s = "aababcabc"
<b>输出：</b>4
<b>解释：</b>总共有 7 个长度为 3 的子字符串："aab"，"aba"，"bab"，"abc"，"bca"，"cab" 和 "abc" 。
好子字符串包括 "abc"，"bca"，"cab" 和 "abc" 。
</pre>

<p> </p>

<p><strong>提示：</strong></p>

<ul>
	<li><code>1 <= s.length <= 100</code></li>
	<li><code>s</code>​​​​​​ 只包含小写英文字母。</li>
</ul>

## 解题思路与复盘

1. **一句话直击本质：** 使用滑动窗口技术遍历字符串，检查每个长度为三的子字符串是否由不同字符组成。

2. **综合思路：**
   - **滑动窗口法：** 所有版本都采用了滑动窗口的思路，通过维护一个长度为三的窗口，逐步滑动检查每个窗口内的字符是否各不相同。
   - **集合判断：** 使用集合来判断窗口内字符是否各不相同，通过比较集合的长度和窗口的长度来实现。

3. **全量伪代码：**
   ```plaintext
   如果字符串长度小于3，返回0

   初始化计数器 cnt 为0
   初始化窗口 win 为字符串的前3个字符

   如果窗口中的字符各不相同，计数器 cnt 加1

   从索引3开始遍历字符串直到结束：
       更新窗口为当前索引及其前两个字符组成的子字符串
       如果窗口中的字符各不相同，计数器 cnt 加1

   返回计数器 cnt 的值
   ```

4. **复杂度：**
   - 时间复杂度：$O(n)$，其中 $n$ 是字符串的长度，因为每个字符只被访问一次。
   - 空间复杂度：$O(1)$，因为使用的额外空间与输入字符串的长度无关，仅用于存储固定大小的窗口和计数器。
